# Zynex AI - Demo Video Script

**Target Duration:** 60 seconds
**Tone:** Professional, Fast-paced, Tech-forward
**Resolution:** 1920x1080 (Horizontal) & 1080x1920 (Vertical for Reels/Shorts)

---

## Scene 1: The Introduction (0:00 - 0:10)
**Visual:** Fade in to the Zynex landing page. Slow zoom in on the main interface. Clean, modern aesthetic.
**Voiceover:** "Meet Zynex. The intelligent, conversational AI interface designed for speed and simplicity. Built with a robust FastAPI backend and a zero-dependency frontend, it's the future of chat."
**On-Screen Text:** ZYNEX - Intelligent Conversational AI

## Scene 2: Zero-Config Setup (0:10 - 0:20)
**Visual:** Cursor clicks the 'Activate Free AI' banner. A subtle highlight circle appears around the click. The UI instantly updates to ready state.
**Voiceover:** "Forget complex setup. With our Zero-Config OAuth, simply click to activate free tier AI models instantly. No .env files required."
**On-Screen Text:** Zero-Config Setup. Instant Access.

## Scene 3: Lightning Fast Responses (0:20 - 0:35)
**Visual:** Cursor clicks into the text area. Types: "Can you explain how the SSE streaming works in Zynex?". Presses enter. The progress bar triggers, and text starts streaming in instantly, token-by-token.
**Voiceover:** "Experience microsecond latency. Zynex uses Server-Sent Events to stream responses in real-time, completely bypassing Virtual DOM diffing for maximum performance."
**On-Screen Text:** True Real-Time Streaming via SSE.

## Scene 4: Markdown & Code Highlighting (0:35 - 0:50)
**Visual:** The AI generates a response that includes a mermaid diagram or code block. The custom markdown compiler renders it beautifully on the fly.
**Voiceover:** "It handles complex markdown, code blocks, and diagrams seamlessly. All locked into a single viewport so your workflow is never interrupted."
**On-Screen Text:** Custom Markdown Compilation.

## Scene 5: Outro & Call to Action (0:50 - 1:00)
**Visual:** Show the UI toggling between light and dark mode, then zoom out slightly as the Zynex logo and GitHub link fade in.
**Voiceover:** "Highly optimized, self-hostable, and entirely open-source. Upgrade your AI experience today."
**On-Screen Text:** Try Zynex on GitHub.
